Use the controls on the left to create a new LED layout for your plane.
Once you have it set up the way you want it, click the Generate Config button
and copy the text in the config.h window and save it as a file called config.h on your PC.
Locate the config.h file in the Arduino sketch folder that you downloaded earlier, and replace
it with this new config file. Now you can proceed to load the new code to the controller by
following the rest of the instructions.
